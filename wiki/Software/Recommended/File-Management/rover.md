## Rover

Rover is a lightweight terminal file manager designed around simplicity, speed, and keyboard-driven navigation. It provides the features needed for everyday file management while remaining small and easy to understand.

Rover follows the Unix philosophy of doing one job well. Rather than replacing the shell, it complements it by making common file operations faster and more convenient. It avoids unnecessary complexity and focuses on responsiveness and ease of use.

Compared to more feature-rich terminal file managers, Rover maintains a balance between functionality and minimalism, making it well suited for users who spend most of their time in the terminal.
