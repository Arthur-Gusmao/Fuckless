## mksh

mksh (MirBSD Korn Shell) is a modern implementation of the Korn Shell, combining POSIX compatibility with a rich interactive environment. It is lightweight, portable, and suitable for both scripting and everyday shell use.

### Features
- Interactive command editing
- POSIX-compatible scripting
- Portable across many operating systems
- Small and efficient

## Philosophy

mksh aims to provide a practical shell that remains fast and reliable while offering the conveniences expected from an interactive shell. It strikes a balance between minimalism and usability.
