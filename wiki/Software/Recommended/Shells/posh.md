## posh

posh (Policy-compliant Ordinary SHell) is a minimal POSIX shell derived from pdksh. It is designed to expose non-portable shell scripts by adhering closely to the POSIX specification.

### Features
 - Strict POSIX behavior
 - Lightweight implementation
 - Minimal interactive features

### Philosophy

Rather than providing convenience extensions, posh encourages portable scripting by rejecting non-standard constructs whenever possible.
