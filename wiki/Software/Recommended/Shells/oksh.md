## oksh

oksh (OpenBSD Korn Shell) is a portable version of the Korn shell implementation developed for OpenBSD. It provides a familiar interactive environment while remaining compact and standards-oriented.

### Features
 - Korn shell compatibility
 - Interactive command editing
 - POSIX-compliant scripting
 - Lightweight and portable

### Philosophy

oksh follows the OpenBSD philosophy of correctness, simplicity, and reliability. It offers a traditional Korn shell experience without unnecessary complexity.
