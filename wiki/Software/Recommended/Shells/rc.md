## rc

rc is the command shell of the Plan 9 operating system. It was designed as a cleaner, simpler alternative to the Bourne shell, featuring a more consistent syntax and treating lists as first-class values.

### Features
 - Clean and consistent syntax
 - Native list handling
 - Simple control structures
 - Lightweight implementation

### Philosophy

rc emphasizes readability and simplicity. Rather than extending Bourne shell semantics, it introduces a more coherent language that reflects the design principles of Plan 9.
