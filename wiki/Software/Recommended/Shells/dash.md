## dash

dash (Debian Almquist Shell) is a lightweight POSIX-compliant shell designed for speed and portability. It implements the POSIX shell standard with minimal extensions, making it an excellent choice for system scripts and environments where startup time and low resource usage are important.

### Features
 - POSIX-compliant
 - Extremely fast startup
 - Small memory footprint
 - Minimal dependencies

### Philosophy

dash focuses on correctness, portability, and performance rather than interactive features. It is commonly used as /bin/sh on many Linux distributions.
